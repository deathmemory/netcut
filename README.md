# NetCut · 网络剪切板

跨设备共享文本、图片和文件的轻量网络剪切板。

## 功能

- 粘贴文本/图片（`Ctrl+V`）或拖放文件即可保存
- 支持文本、图片、任意文件
- 一键复制文本/图片，文件直接下载
- 数据持久化存储，30 秒自动同步
- 同一局域网内所有设备均可访问

## 使用

```bash
npm install
npm start
```

浏览器打开 `http://localhost:3100`，局域网其他设备访问 `http://<本机IP>:3100`。

## 技术栈

- **后端**：Node.js + Express + Multer
- **前端**：原生 HTML / CSS / JavaScript
- **存储**：JSON 文件（文本/图片）+ 本地磁盘（文件）
